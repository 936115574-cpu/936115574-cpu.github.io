# GitHub Pages Academic Homepage

This is a ready-to-upload personal academic website for GitHub Pages.

## How to Use

1. Create a public GitHub repository named:

   ```text
   your-username.github.io
   ```

2. Upload all files in this folder to the repository root.

3. Go to:

   ```text
   Settings -> Pages
   ```

4. Set:

   ```text
   Source: Deploy from branch
   Branch: main
   Folder: /root
   ```

5. Replace the placeholders:

   - `Your Name`
   - `your.email@example.com`
   - Google Scholar / ORCID / ResearchGate links
   - publication entries
   - project details
   - `files/CV.pdf`

## Recommended Positioning

Use this website to emphasize:

- Construction Management
- BIM-based construction coordination
- Digital Twin in Construction
- Construction Safety Training
- China State Construction project experience
- Engineering education and doctoral research potential

## SEO Notes

Keep the page in English. Use the keywords naturally in page titles, headings, project descriptions, and publication descriptions.

# GitHub Pages 学术主页模板

这是一个可以直接上传到 GitHub Pages 的个人学术主页模板，适合用于博士申请、导师套磁、科研展示和工程项目展示。

## 使用步骤

1. 在 GitHub 创建公开仓库，名字必须是：

   ```text
   你的用户名.github.io
   ```

2. 把本文件夹里的所有文件上传到仓库根目录。

3. 进入仓库：

   ```text
   Settings -> Pages
   ```

4. 设置：

   ```text
   Source: Deploy from branch
   Branch: main
   Folder: /root
   ```

5. 等待 1 到 3 分钟，打开：

   ```text
   https://你的用户名.github.io
   ```

## 必改内容

- 把所有 `Your Name` 改成你的英文姓名。
- 把 `your.email@example.com` 改成你的正式邮箱。
- 把 `_config.yml` 里的 `url` 改成你的 GitHub Pages 地址。
- 把 `files/CV.pdf` 换成你的英文 CV，文件名保持 `CV.pdf`。
- 在 `publications.html` 里加入真实论文、会议、工作论文。
- 在 `projects.html` 里加入真实工程项目、职责和成果。
- 在 `contact.html` 里替换 Google Scholar、ORCID、ResearchGate、LinkedIn 链接。

## 博士申请定位建议

主页关键词建议保留或自然强化：

- Construction Management
- BIM-based construction coordination
- Digital Twin in Construction
- Digital Construction
- Construction Safety Training
- China State Construction
- Engineering Education

## 项目页写作建议

项目页不要只写“参与项目”，要写成：

- Role: 你的角色
- Organization: 单位
- Contribution: 你做了什么
- Outcome: 带来了什么结果
- Research Translation: 这个工程经验如何转化为博士研究问题

这样更适合欧洲和中东研究型博士导师快速判断你的工程经验是否能转化为研究潜力。
